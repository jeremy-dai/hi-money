# Hi Money - 个人财富管理微信小程序

一个基于"四分法"理财模型的个人财富管理微信小程序，帮助用户通过教育引导和智能规划，实现更好的财务管理。

## 功能特点

### 1. 财富之旅引导
- 欢迎页面介绍核心功能
- 引导用户完成初始设置流程

### 2. 智能财富分配
基于"四分法"模型的资金分配：
- **生活开支 (50%)** - 精明消费，优化日常开支
- **投资增长 (25%)** - 着眼未来，让资金为你工作
- **稳定基金 (15%)** - 构筑基石，建立安全网
- **快乐基金 (10%)** - 投资快乐，享受当下

### 3. 可视化财务仪表盘
- 月度财务规划展示
- 年度资金分配饼图
- 各类别详细金额计算

### 4. 教育内容
每个财富类别都包含详细的教育指导：
- **精明消费**：需求vs欲望分析、关键开支优化、消费决策规则
- **投资增长**：资产选择、税收优惠账户、投资流程
- **稳定基金**：应急基金、保险规划、定期审查
- **快乐基金**：自由消费、快乐目标、平衡当下与未来

## 核心功能说明

### 数据存储
使用微信小程序的本地存储 API (`wx.setStorageSync` / `wx.getStorageSync`) 保存：
- 月收入
- 财富分配比例
- 设置完成状态

### 智能比例调整
财富分配页面实现了智能滑块调整机制：
- 拖动任意滑块时，其他滑块自动按比例调整
- 确保四个类别总和始终为 100%
- 平滑的用户体验

### 可视化展示
- 使用 Canvas 绘制年度资金分配饼图
- 彩色卡片展示各类别金额
- 渐变背景增强视觉效果

## 快速开始

### 1. 环境准备
- 安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号（可使用测试号）

### 2. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `hi-money` 目录
4. 填入 AppID（可使用测试 AppID）

### 3. 运行预览
- 点击"编译"按钮即可预览
- 可以使用模拟器或真机调试

## 使用流程

1. **欢迎页** - 了解应用功能，点击"开始财富之旅"
2. **输入月收入** - 输入税后月收入
3. **设置分配比例** - 调整四个类别的分配比例（默认50/25/15/10）
4. **查看财务规划** - 在仪表盘查看月度规划和年度分配
5. **学习理财知识** - 点击任意类别卡片查看详细教育内容
